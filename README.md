# TaskFlow 🚀

🔗 **[taskflow-jose.netlify.app](https://taskflow-jose.netlify.app)**

Gerenciador de tarefas construído com React, TypeScript e Tailwind CSS.

## Funcionalidades

- ✅ Adicionar tarefas com título e prioridade (Baixa, Média, Alta)
- ✅ Marcar tarefas como concluídas
- ✅ Remover tarefas
- ✅ Contador de tarefas pendentes
- ✅ Persistência local com `localStorage`
- ✅ Suporte a tecla Enter para adicionar tarefa

## Tecnologias

- [React 19](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Vite](https://vite.dev/)

## Como rodar localmente

```bash
# Instalar dependências
npm install

# Iniciar o servidor de desenvolvimento
npm run dev
```

Acesse `http://localhost:5173` no navegador.

## Scripts disponíveis

| Comando | Descrição |
|---|---|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera o build de produção |
| `npm run preview` | Pré-visualiza o build de produção |
| `npm run lint` | Executa o ESLint |
